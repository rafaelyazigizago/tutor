BLOCO DE CONTEXTO PARA O PRÓXIMO ASSISTENTE
════════════════════════════════════════════════════════════

O projeto Tutor AIOS é um professor pessoal de IA local que ingere fontes de
conhecimento (PDFs, futuramente vídeos), gera currículos e conduz aulas
conversacionais em linguagem natural. Desenvolvido por Rafael em Python 3.12.

O projeto estava rodando no Windows 11 (C:\AI\Tutor) com Ollama + qwen2.5:7b,
ChromaDB e sentence-transformers. Backup completo salvo no Google Drive
(ProjTutor_Vrs1_11Mar26_1016.zip). Pipeline ponta a ponta validado e funcional.

A partir desta sessão, o ambiente migra para Linux Zorin 18 num Lenovo IdeaPad
com i7-10750H, 8 GB RAM e GTX 1650. Por limitação de RAM, o projeto adota
estratégia multi-LLM com APIs online: Gemini Flash 2.0 (ingestão), Groq/Llama 3
(conversa), GPT-4o mini (currículo), Claude (raciocínio). Nenhuma chave de API
criada ainda. Ambiente Zorin não verificado ainda.

Estamos na fase de análise de publicações para planejar o v2.0 antes de retomar
o código. Principais decisões v2.0 registradas: Skills como PersonaEngine,
ResearchAgent com protocolo de pesquisa profunda, Oracle Cloud como servidor,
modo Mind Map e Glossário no LessonGenerator, aulas por capítulo do documento.
